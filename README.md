# macOS-D-day

 * 특정 날짜를 기준으로 상단 Dock Bar에 디데이를 표시해줍니다.
 * tested only on MacOS Monterey `12.5.1`, Apple M1 Pro

## stack
 * python 3.11
 * rumps 0.3.0
 * py2app 0.28.7

## quick start
```
python3 main.py
```

## quick build
```
python3 setup.py py2app -A
```
